Anleitung für die Nutzung von dbDemo

1. Importieren Sie den Ordner dbDemo/demo in eclipse oder eine ähnliche Entwicklungsumgebung.
2. Führen Sie die main aus.
3. Führen Sie Abfragen aus z.B. über den Terminal mit 'wget http://localhost:8080/betriebsstelle/aamp' 
oder über den Browser mit 'http://localhost:8080/betriebsstelle/aamp'
